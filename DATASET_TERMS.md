# SCCGBench Dataset Terms

SCCGBench is released for academic research purposes only.

The dataset contains service-call code snippets mined from public GitHub repositories and aligned with RapidAPI documentation. The original copyright and any original licenses of these third-party code snippets remain with their respective repository owners. The SCCGBench authors do not relicense third-party code snippets.

All embedded real third-party credentials, including RapidAPI keys, OpenAI keys, Google API keys, GitHub tokens, and other secrets, have been redacted and replaced with `<REDACTED_*>` placeholders.

Users of this dataset are responsible for:

1. using the dataset only for academic and non-commercial research unless they have obtained additional permissions;
2. respecting the licenses and terms of the original GitHub repositories;
3. not attempting to recover, infer, or misuse any redacted credentials;
4. not using the dataset to access paid, private, or restricted APIs without authorization;
5. not using the dataset in a way that violates the terms of RapidAPI, GitHub, or any original service provider.

The metadata, statistics, annotations, redacted intents, construction scripts, and evaluation outputs created by the SCCGBench authors are provided for research and reproducibility purposes.

If you use SCCGBench in academic work, please cite the corresponding paper and this repository.

# SCCGBench 数据集使用条款

SCCGBench 数据集仅供学术研究使用。

本数据集包含从公开 GitHub 仓库挖掘得到的服务调用代码片段，并与 RapidAPI 文档进行对齐。这些第三方代码片段的原始版权及原始许可证仍归对应仓库作者所有。SCCGBench 作者不对第三方代码片段进行重新授权。

所有嵌入的第三方真实凭证，包括 RapidAPI Key、OpenAI Key、Google API Key、GitHub Token 及其它密钥，均已脱敏并替换为 `<REDACTED_*>` 占位符。

数据集使用者应负责：

1. 仅将数据集用于学术和非商业研究，除非已获得额外授权；
2. 遵守原始 GitHub 仓库的许可证和使用条款；
3. 不尝试恢复、推断或滥用任何已脱敏凭证；
4. 不在未经授权的情况下使用数据集访问付费、私有或受限 API；
5. 不以违反 RapidAPI、GitHub 或任何原始服务提供方使用条款的方式使用数据集。

SCCGBench 作者创建的元数据、统计信息、标注、脱敏意图、构建脚本和评估输出仅用于研究和复现目的。

如果你在学术工作中使用 SCCGBench，请引用对应论文和本仓库。
